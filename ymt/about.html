Welcome to Our TV Channel App

This application provides access to various TV channels across different categories and languages.

Features

  • Browse channels by category
  • Filter channels by language
  • Direct channel access using number keys
  • Easy navigation using D-pad controls

Controls

  • Up/Down/Left/Right: Navigate through channels
  • Enter: Select
  • Number keys (0-9): Direct channel access
  • Escape/Back: Return to previous screen
